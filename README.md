# garbage-recovery
废品回收平台，支持客户和员工端
"用户端：用户等级/用户收入/余额/微信红包提现/申请加入我们

/订单/个人设置"

用户端：GPS定位/显示当前区县对应的废品价格、活动、广告等

"员工端：抢单模式/地图导航/信用评级/保证金充值

/余额提现/周边废品站点导航/个人设置"

站点端：会员统计/财务统计/员工热点地图

站点端：订单管理/用户管理/站点充值
